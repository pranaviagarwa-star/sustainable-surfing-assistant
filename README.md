# 🌱 Sustainable Surfing Assistant
A browser extension that helps users make eco-friendly shopping decisions.
## 🚀 Features
- Detects product pages automatically
- Analyzes sustainability impact
- Shows Eco Score (out of 10)
- Provides warnings (deforestation, fast fashion, etc.)
- Suggests eco-friendly alternatives
## 🛠️ Tech Used
- JavaScript
- Chrome Extension API
## 💡 Goal
To reduce environmental impact by making users aware of sustainable choices while browsing.
